## JS Math
원주률인 `const num = 3.1415926535;`를 간단히 예시로 써보자.

#### Math.floor() : 소수점 이하를 버림한다.

```javascript
Math.floor(num);  // 3
```

<br/>

#### Math.ceil() : 소수점 이하를 올림한다.

```javascript
Math.ceil(num);   // 4
```

<br/>


#### Math.round() : 소수점 이하를 반올림한다.

```javascript
Math.round(num);  // 3
```

<br/>
